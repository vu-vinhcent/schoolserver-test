Complexity

What is a strong password?
  • Contains a combination of both uppercase and lowercase letters, digits, and symbols.
  • Has at least 8 characters.
  • Doesn't contain words that can be found in a dictionary.
Why do we need strong passwords?
  • Brute Force Method: algorithm that tries all possible character combinations to find a password
    • A password that has 8 lowercase letters has a total of 208,827,064,576 possibilities.
    • A password that has 8 lowercase and uppercase letters has a total of 2,821,109,907,456 possibilities.
    • In general, the more types of characters and the more characters you have in your password, the better.
    • There is no limitation to the brute force method, but eventually, the time it takes to crack the password will become unreasonable.
  • Dictionary Attack Method: algorithm that checks to see if password can be found in a dictionary
    • If the word is in the dictionary, it will be found, regardless of how complex or how long it is.
    • Provided the word is in the dictionary, it is much faster than brute force.
    • The limitation is that it only works if the password is in the dictionary.

Default Passwords

What are default passwords?
  • Manufacturers provide default usernames and passwords to make the initial setup of a device simpler.
  • Default credentials are also used whenever a device is reset back to factory settings.
  • Default passwords are usually simple, and not unique to each individual device.
  • Default passwords can be found in the instruction manuals, or on the device itself. There are many websites that list the default credentials for mnay devices.
  • Attackers can easily obtain default passwords and identify internet-connected target systems.

What should you do?

Always change default passwords as soon as possible on any internet-connected device, absolutely before you connect to the internet.
